1) Choose whether to record or train.
2) Press 1 to record words.
2) Select a word for which you want to record.
3) Data will be saved automatically into Folder hierarchy “saved_recordings/words_name/”.
4) Counter files for each of the recorded words is kept to keep track of current iteration and file is saved at : “saved_recordings/iteration_wordname.txt”.
5) Counter for all files can be reset using 999 as input choice in main menu to start recording fresh and saved data gets overwritten.
6) Press 2 to train all the saved recordings.
